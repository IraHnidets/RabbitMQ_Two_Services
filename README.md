# RabbitMQ_Two_Services
1. Розгорнути RabbitMQ за допомогою Docker або локальної інсталяції.  2. У відповідності до варіанту вибраного у лабораторній роботі №1 реалізувати два  сервіси:  o Producer: приймає вхідні дані від користувача та відправляє їх у RabbitMQ.  o Consumer: отримує повідомлення, обробляє їх  та зберігає у файл.
